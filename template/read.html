Comment déboguer une erreur JWT ?

posée par Ali · il y a 2h
jwt authentification node.js

J'obtiens une erreur lors de la vérification de mon token JWT. Voici le code utilisé et le message d'erreur retourné. Comment puis-je résoudre ce problème ?

const token = req.headers['authorization'];
try {
  const decoded = jwt.verify(token, 'secret');
} catch (err) {
  console.log(err);
}

2 Réponses

5 votes

Assurez-vous que le token envoyé dans l'en-tête est bien formé et que la clé secrète utilisée pour le signer correspond à celle utilisée pour la vérification.

répondu par Sara · il y a 1h
2 votes

Vérifiez également que le token n'a pas expiré et que le format de l'en-tête Authorization est correct : Bearer <token>.
